# Abalone Age Prediction: Comparing CART Regression and Random Forest Models

Abalones are marine snails found in warm seas worldwide and are economically valuable as food and for their decorative shells. Their market price largely depends on age — older abalones are typically more valuable.

Age is traditionally estimated by counting the number of rings inside the shell (similar to tree rings). However, this process requires cutting and preparing the shell, making it time-consuming and sometimes inaccurate. Because of this, researchers explore alternative methods to estimate age using physical characteristics such as sex, length, height, and weight.

This project aims to predict the age of abalones using CART regression and Random Forest models, implemented in R.

The dataset, sourced from the UCI Machine Learning Repository, contains 4,177 samples with eight predictor variables (Sex, Length, Diameter, Height, Whole_weight, Shucked_weight, Viscera_weight, Shell_weight) and one target variable (Rings), which serves as a proxy for age.
